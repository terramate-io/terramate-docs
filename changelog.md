# Changelog

This changelog highlights important changes across docs structure and key Terramate capabilities.

For full release artifacts, see [GitHub Releases](https://github.com/terramate-io/terramate/releases).

## 2026-02

### Documentation

- Introduced a new top-level information architecture with dedicated **Get Started**, **Guides**, and **Cloud API Reference** entry points.
- Added use-case guides:
  - Manage Terraform at scale
  - Set up drift detection
  - Migration guides from Terragrunt and Terraform-only workflows
- Added foundational docs:
  - [Glossary](/glossary)
  - [Changelog](/changelog)
  - LLM index (`/llms.txt`)
- Reorganized sidebar into grouped domains:
  - Platform
  - Orchestration & Automation
  - Cloud Features
  - Integrations
  - Administration
  - Reference

### API Reference

- Expanded Cloud API reference pages with a consistent endpoint-oriented template:
  - endpoint operation
  - request fields
  - request examples
  - response examples
  - error model pointers
