# Code Generation Concepts

Terramate uses generation over abstraction so output stays native and compatible with your existing IaC ecosystem.

## Why this approach

- Keep stacks DRY
- Preserve compatibility with Terraform/OpenTofu tooling
- Avoid lock-in through native generated output
- Support static analysis and policy tooling
