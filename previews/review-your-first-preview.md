# Review Your First Pull Request Preview

This tutorial guides you through running a preview workflow and inspecting results in Terramate Cloud.

## Steps

1) Configure a preview workflow in your CI/CD.
2) Open a pull request with infrastructure changes.
3) Let the workflow publish preview data to Cloud.
4) Inspect stack-level plan summaries in the PR preview view.

## Expected result

- Preview metadata and stack impact are visible in Terramate Cloud and linked SCM views.
