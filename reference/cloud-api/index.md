# Cloud API

The Terramate Cloud API is resource-oriented and designed for automation workflows around stack visibility, previews, deployments, drift, and alerts.

## Base URL

```text
https://cloud.terramate.io/api
```

The Cloud API is evolving and endpoint coverage is documented incrementally.
